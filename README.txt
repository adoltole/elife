DESCRIPTION

Elife is a drush script designed to automate certain drupal repetitive tasks.

=========================================

INSTRUCTIONS

1- Create a file called "vars.php" using "default.vars.php" included in the repository (cp default.vars.php vars.php) and modify the content with your data.
2- Copy the default preproduction vitualhost "defaultapache-pre.conf" to /etc/apache2/sites-available/. This template is designed for a preproduction site where authentication is required.
